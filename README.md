# 🛍️ Seasonal Style Hub

A modern **Full Stack E-Commerce Web Application** that helps customers browse, filter, and purchase clothing collections based on different seasons. The application provides a smooth shopping experience with product browsing, shopping cart management, user authentication, and online checkout.

---

## 📖 Project Overview

Seasonal Style Hub is designed to make seasonal shopping easier by organizing clothing collections according to **Spring, Summer, Autumn, and Winter**. Customers can filter products, add them to the shopping cart, and complete purchases through a simple checkout process.

The project is developed using **HTML, CSS, JavaScript, PHP, and MySQL** with **XAMPP** as the local server.

---

## ✨ Features

- 🏠 Attractive Home Page with Image Slider
- 👕 Seasonal Clothing Collections
- 🔍 Filter Products by:
  - Season
  - Category
  - Price
- 🛒 Shopping Cart
- ❤️ Wishlist Support
- 👤 User Login
- 💳 Checkout Page
- 📱 Responsive Design
- 🗄️ MySQL Database Integration
- ⚡ Dynamic Product Loading using PHP

## 🚀 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Pooja17-p/Seasonal-Style-Hub.git
```

### 2. Move the Project

Copy the project folder into

```
xampp/htdocs/
```

### 3. Start XAMPP

Start

- Apache
- MySQL

### 4. Create Database

Open

```
http://localhost/phpmyadmin
```

Create a database named

```
seasonal_style_hub
```

Import the SQL file.

### 5. Run the Project

Open

```
http://localhost/Seasonal-Style-Hub/
```
